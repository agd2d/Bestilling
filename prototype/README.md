# Prototype

Dette er et separat visuelt testmiljø til varebestillingsmodulet.

Formålet er at gøre logikken synlig tidligt, før vi bygger den rigtige Next.js-implementering ind mod `Kundedashboard`.

## Indeholder

- overblik for varebestilleren
- kundebestillingsliste
- ordredetalje med linjer, labels og noter
- leverandørbestillinger
- mailindbakke
- varekatalog

Åbn `index.html` i en browser for at gennemgå flowet visuelt.
